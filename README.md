# Y
Welcome to the *Y* language

In this document, you will find all of the helpful resources in order to navigate this unique language.

Y is a minimal pipeline-oriented expression language.

It is built to be some of the quickest programming you'll encounter, with the potential cost of syntax that is a little difficult to read.

It is similar to functional programming languages, but contains some pieces that are not too tradional.

If you are just learning programming, this language is not recommended.

*This language is not yet completed*

This language was built as a side project by @corcordan.
